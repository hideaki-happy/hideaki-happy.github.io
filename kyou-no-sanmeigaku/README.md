# 今日の算命学 App Store公開用ページ

GitHub Pagesで公開するための静的ページです。

## ページ

- `index.html`: 入口ページ
- `support.html`: App Store ConnectのサポートURLに使用
- `privacy.html`: App Store ConnectのプライバシーポリシーURLに使用
- `styles.css`: 共通スタイル

## 公開前に変更する項目

`support.html` の問い合わせメールアドレスを、実際に受け取れるメールアドレスへ変更してください。

現在の仮メールアドレス:

```text
support@example.com
```

## App Store Connectに入力するURL例

リポジトリ名を `kyou-no-sanmeigaku-support` にした場合の例です。

```text
サポートURL:
https://hideaki-happy.github.io/kyou-no-sanmeigaku-support/support.html

プライバシーポリシーURL:
https://hideaki-happy.github.io/kyou-no-sanmeigaku-support/privacy.html
```
